# ANEMOS 3.0

Nuova architettura di ANEMOS.

Obiettivi principali:

- Costruzione libera delle respirazioni
- Tre volumi essenziali:
  - Addome
  - Torace inferiore
  - Torace superiore
- Organizzazione simultanea, sequenziale
- Motore di coerenza non bloccante
- Libreria delle respirazioni
- Respirazione guidata
- Anemoscopio
- Architettura modulare
